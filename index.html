    About Toronto Walks

    Welcome! My name is Gary Smith. I am a web developer and GIS student who has lived in Toronto for over 25 years. I've created this app to help locals and visitors alike explore some of the fascinating sights around the city I call home.

    Questions? Suggestions?

    I'd be happy to hear from you! If you have questions or concerns about this app, or suggestions and ideas for additional walks, please send me an email using my portfolio site.

    Sources & Credits

    The writing and image content presented in this app is the property of the author, with the exception of several placeholder public domain photos, which are credited inline.

    The interactive map functionality for this app was implemented using the Leaflet.js Javascript library, with OpenStreetMap from CARTO.

    Additional map layer data for sidewalks, greenspaces, building outlines and water features were obtained from the City of Toronto Open Data Portal.

    The source code for this app can be viewed on GitHub.
